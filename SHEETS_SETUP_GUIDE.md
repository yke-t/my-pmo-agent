# Google Sheetsの構造改善手順書

## ⚡ クイックスタート（5分で完了）

### Issue シートの改善

1. **[スプレッドシートを開く](https://docs.google.com/spreadsheets/d/1C6ua596iFVCG2fx6YnYviqthrF3K4EYYxUPJhNUUt7A/edit)**

2. **Issueシート**を選択

3. **列を追加**（G列～K列に以下を追加）:

| 列 | カラム名 | データ型 | 設定 |
|----|---------|---------|------|
| G | ベンダー名 | テキスト | - |
| H | 優先度 | プルダウン | `緊急,高,中,低` |
| I | 期限 | 日付 | YYYY-MM-DD形式 |
| J | 影響範囲 | プルダウン | `全体,特定ベンダー,限定的` |
| K | 更新日 | 日付 | YYYY-MM-DD形式 |

4. **プルダウン設定**:

   **優先度（H列）**:
   - H1セル（ヘッダー行）以外を選択
   - データ → データの入力規則
   - 条件: リストを範囲で指定 → 手動で入力: `緊急,高,中,低`
   - 保存

   **影響範囲（J列）**:
   - 同様に: `全体,特定ベンダー,限定的`

---

### Schedule シートの改善

1. **Scheduleシート**を選択

2. **列を追加**（A列に挿入、その他を追加）:

| 列 | カラム名 | データ型 | 設定 |
|----|---------|---------|------|
| A | ID | 数値 | 1から連番 |
| （タスクは元のA列→B列に移動） |
| C | ベンダー名 | テキスト | - |
| D | 担当者 | テキスト | - |
| （開始予定、終了予定は従来通り）|
| G | ステータス | プルダウン | `未着手,進行中,停滞,完了,保留` |

3. **ステータスのプルダウン設定**:
   - G列（ヘッダー以外）
   - データの入力規則: `未着手,進行中,停滞,完了,保留`

## 📝 最終的なシート構造

### Issue Log

```
ID | 起票日 | カテゴリ | 内容 | 担当者 | ステータス | ベンダー名 | 優先度 | 期限 | 影響範囲 | 更新日
```

### Schedule

```
ID | タスク | ベンダー名 | 担当者 | 開始予定 | 終了予定 | ステータス | 進捗率 | メモ
```
